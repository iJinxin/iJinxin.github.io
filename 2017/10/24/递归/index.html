几个经典的递归问题

走楼梯

一共10级楼梯，每次可以走一步或者两步，求一共多少种走法?

Thinking:要想走到第n级台阶，一共有两种走法：

  • 从n-2级台阶迈两步
  • 从n-1级台阶迈一步

对于如何走到 n-2 和 n-1 级台阶，也是分了以上两种情况，以此类推。
那么，这个问题可以简化为 step(n) = step(n-2) + step(n-1) 。

不难发现 step(0) = 0，step(1) = 1, step(2) = 2。

细胞分裂

一个细胞，一个小时分裂一次，生命周期是三个小时，求n个小时后，有多少细胞?

Thinking:细胞在其生命周期分为三种状态：

  • stateA: 刚分裂 –> 由前一个小时的stateA, stateB, stateC 分裂组成
  • stateB: 分裂一小时 –> 由前一小时stateA细胞生长组成
  • stateC: 分裂两小时 –> 有前一小时stateB细胞生长组成
  • stateD: 死亡 –> 由之前累积的stateD和一小时前stateC细胞死亡组成

在任意时期，容器内的细胞数量为 stateA + stateB + stateC。显然 stateA(0) = 1, stateB(0) = 0, stateC(0) = 0 。
